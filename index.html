Ice Scream
Won 1st Place in Stanford CS248 Videogame Competition
with judges from Blizzard, Riot, and PocketGems
An action/strategy game where you use ice cream to defeat evil but lactose intolerant monsters! This was the first 3D game I've ever made, and I designed all the assets/characters/animations in Blender. I made this game within a span of 10 weeks, by coding with C# in Unity3D. You can read about fun details/challenges in the game report here!
Penguin Tag
Used deep learning to create opponent AI
Have you ever had a snowball fight combined with a game of tag? Well now you can! I created the opponent AI using neural networks in Tensorflow - and trained it on data from my own games, this way, it plays like a human player would.

Quiz4
Online multiplayer game
Connect4 with a twist - you can only make a turn if you're the first to unscramble the word! This game has online realtime multiplayer support, so you can play with your friends across the world! I coded the server logic using Node.js and socket.io, and created the game using Javascript and React.
Computer Eyes
Research in TJHSST Computer Systems Lab
Over the course of a year, I created software that would allow a user to control a computer using only their eyes. I used the Tobii EyeX eyetracker and C++ to develop a program that completely overrode the computer mouse - making computing more accessible for those with limited mobility, and more sanitary as well.
Moonlens
Stanford TreeHacks 2017 - VR Grand Prize Winner
Me and my dorm-mates noticed that Virtual Reality was often too expensive. To fix that, we built Moonlens, which is a three-step platform that makes VR cheaper and more accessible. We built everything in just 36 hours. I made all the graphics and the mobile app for the project.
Positivity
DCHacks 2015 - Best Overall App Winner
I wanted to build a community for PTSD victims, so I created Positivity. It allows members to post uplifting messages and uses a system of upvotes to determine which posts are most effective. Users can choose to be notified if a post reaches a certain threshold of upvotes so that they never miss top posts.
